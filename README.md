# Chrismacolor/homebrew-tap

Homebrew tap for [Margins](https://github.com/Chrismacolor/margins) — a
lightweight, native macOS Markdown reader.

## Install

```bash
brew install --cask Chrismacolor/tap/margins
```

Upgrades come through `brew upgrade` — there's no in-app updater.

## Casks

- [`margins`](Casks/margins.rb) — the Margins app.
